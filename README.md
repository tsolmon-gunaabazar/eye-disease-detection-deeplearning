# eye-disease-detection-deeplearning
**Eye Disease Detection from OCT Scans using DL**

Eye diseases such as diabetic macular edema (DME), choroidal neovascularization (CNV), and drusen are the leading cause of vision loss around the world. Many of these diseases affect the retina and can be detected using medical imaging techniques such as Optical Coherence Tomography (OCT). OCT scans produce high-resolution cross-sectional images of the retina that allow doctors to observe structural changes in eye tissue (Kermany et.al.,2018). However, analyzing these scans requires specialized expertise and can be time-consuming when large numbers of images must be examined. Because of this, automated classification systems that can assist with identifying retinal diseases from OCT images could help reduce the workload for clinicians and support earlier detection of vision-threatening conditions. The goal of this project is to develop a deep learning model that can automatically classify OCT images into four categories: *Normal*, *CNV*, *DME*, and *Drusen*. The input to the model is a grayscale OCT image, and the output is the predicted disease class associated with that image. The task can therefore be formulated as an image classification problem where the model maps an OCT image to one of four diagnostic labels. This problem is well suited for deep learning because it is fundamentally an image classification task. Convolutional Neural Networks (CNNs) are designed to learn visual patterns directly from images by identifying features such as edges, textures, and structural shapes. In OCT scans, diseases often appear as subtle structural changes in the retinal layers, which makes CNNs particularly effective at detecting these patterns automatically. In this project, deep learning models will be used to learn these visual features from OCT images and classify different types of retinal disease.
